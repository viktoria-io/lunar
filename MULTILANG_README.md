# Система мультиязычности для сайта

## Обзор

Реализована полноценная система мультиязычности с поддержкой трех языков:
- **RU** - Русский (по умолчанию)
- **EN** - English  
- **UA** - Українська

## Структура файлов

### Основные файлы:
- `js/translations.js` - Файл со всеми переводами на 3 языка
- `js/multilang.js` - Основной скрипт управления мультиязычностью
- `css/style.css` - Добавлены стили для активных состояний кнопок

### HTML изменения:
- Всем текстовым элементам добавлены атрибуты `data-i18n`
- Формам добавлены атрибуты `data-i18n-placeholder` и `data-i18n-value`
- Переключатели языков имеют атрибуты `data-lang="ru|en|uk"`

## Как это работает

### 1. Автоматическая инициализация
При загрузке страницы система:
- Проверяет `localStorage` для сохраненного языка
- Если нет сохраненного языка, устанавливает RU по умолчанию
- Применяет переводы ко всем элементам с `data-i18n`

### 2. Переключение языков
- Два переключателя: в шапке (ПК) и плавающая кнопка (мобильные)
- Синхронизация между переключателями
- Сохранение выбора в `localStorage`
- Мгновенное переключение без перезагрузки

### 3. Активные состояния
- Текущий язык подсвечивается розовым цветом
- Текущий язык скрывается из дропдауна
- Добавлен класс `.active` для выбранного языка

## Использование в коде

### Добавление нового перевода:
1. В `translations.js` добавьте ключ для каждого языка:
```javascript
ru: {
    my_new_text: "Новый текст на русском"
},
en: {
    my_new_text: "New text in English"
},
uk: {
    my_new_text: "Новий текст українською"
}
```

2. В HTML добавьте атрибут:
```html
<p data-i18n="my_new_text">Текст по умолчанию</p>
```

### Для форм:
```html
<input data-i18n-placeholder="form_name" placeholder="Name">
<button data-i18n-value="submit_text">Submit</button>
```

### Программное переключение:
```javascript
// Получить текущий язык
const currentLang = window.multiLang.getCurrentLanguage();

// Переключить язык
window.multiLang.changeLanguage('en');

// Получить перевод
const text = window.multiLang.t('my_key');
```

    // Обновляет ответы с HTML-разметкой

- Отдельный плавающий переключатель для мобильных устройств

- Специальные стили для мобильных активных состояний

- Сохранена обратная совместимость со старым `content_data.js`

## Тестирование

1. Откройте сайт в браузере
2. Переключите язык через дропдаун в шапке
3. Проверьте мобильную версию (размер < 991px)
4. Убедитесь, что выбор сохраняется после перезагрузки
5. Проверьте все секции сайта на корректность переводов

## Добавление нового языка

1. В `translations.js` добавьте новый объект языка:
```javascript
fr: { // Французский
    // все переводы
}
```

2. В HTML добавьте новую опцию:
```html
<div class="language-option" data-lang="fr">FR</div>
```

3. В `multilang.js` обновите `currentLangTexts` если нужно

## Troubleshooting

### Если переводы не применяются:
- Проверьте подключение `translations.js` и `multilang.js`
- Убедитесь что атрибуты `data-i18n` корректны
- Проверьте консоль браузера на ошибки

### Если переключатель не работает:
- Проверьте что у элементов есть `data-lang` атрибуты
- Убедитесь что CSS стили активных состояний применяются
- Проверьте мобильную/десктоп версию отдельно
